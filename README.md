# PermuteDDS
The official implementation of our paper:  
PermuteDDS: A permutable feature fusion network for Drug-Drug Synergy prediction

![](PermuteDDS.jpg)

# Requirements
* python: 3.7
* pytorch: 1.11
* rdkit: 2020.09.1
* numpy: 1.18
* tmap: 1.0

# Run the code
Select the dataset in main.py
```
python main.py
```
